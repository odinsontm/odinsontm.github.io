Number Ninja

This 2D platformer was created as a part of CMPT 322. The goal of the class was to simulate a real-world client-building job. Our assignment was to create a entertaining yet informative game that helped 2nd and 3rd graders with their math skills. So, we made a 2D platformer where the user acted as a ninja running away from a dragon. Along the way, there are jumps where the user is prompted with a math question. To land the jump, the user must answer the question correctly. If the user completes all the questions and makes all the jumps, they win the level.

This project gave me experience in software design and development as it applies to game development. It also gave me experience working as a team, coordinating work, and solving issues as they arose. This gave me effective practice with communication amoung team members and our pretend client (our professor). It gave our team an oppourtunity to apply learning in a creative and engaging way. As well as practice global resposibility by creating a engaging learning tool for 2nd-3rd grade teachers.

Technologies Used:

  • Unity
  • C#
